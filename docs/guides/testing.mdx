# Testing

The SDK ships with a robust test setup using **Vitest**. Tests cover copy/paste/cut, grouping, transformations, and complex scenarios.

## Run Tests

```bash npm2yarn
npm test
```

## UI Runner

```bash npm2yarn
npm run test:ui
```

## Coverage

```bash npm2yarn
npm run test:coverage
```

For details, see the repository guides: `QUICK_START_TESTING.md` and `TESTING_GUIDE.md`.
