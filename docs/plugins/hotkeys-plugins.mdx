# Hotkeys Plugins

Keyboard shortcuts for camera and nodes.

## Usage

```ts
import { CameraHotkeysPlugin, NodeHotkeysPlugin } from "@flowscape-ui/core-sdk";

const engine = new CoreEngine({
  container,
  plugins: [new CameraHotkeysPlugin(), new NodeHotkeysPlugin()],
});
```

Common shortcuts include zoom in/out, pan, duplicate, delete, etc.
